Playful Sparkle - SEO Pack Extension for OpenCart 4

The Playful Sparkle - SEO Pack is a comprehensive SEO extension designed to enhance your OpenCart 4 webshop’s search engine visibility. This extension integrates structured data, rich snippets (Structured Data Markup (LD+JSON)), and metadata tags (Dublin Core, Open Graph, and Twitter) into your webshop’s header, ensuring better performance in search results. It supports multi-store and multi-language setups, allowing for individual SEO customization for each store.

Demo frontend - https://opencart4.playfulsparkle.com/
Demo backend - https://opencart4.playfulsparkle.com/admin/

Key Features

- Structured Data Markup (LD+JSON): Supports rich snippets, including product information, price, availability, ratings, reviews, organization, and return policies.
- Dublin Core Metadata: Adds essential metadata for content description.
- Open Graph Metadata: Generates Open Graph tags for effective product sharing on social platforms like Facebook.
- Twitter Metadata: Incorporates Twitter card metadata for products, including pricing and availability.
- Multi-store and Multi-language Support: Customize metadata and structured data for each store and language.
- Rich Customization Options: Personalize your store's name, owner details, description, and logo.
- Link Tag Generation: Automatically adds `<link>` tags for language support, improving SEO across multiple languages.
- Full Control: Configure rich snippets for stock status, shipping methods, return policies, and more.

Installation Guide

1. Download the "Playful Sparkle - SEO Pack" Extension Package.
2. Download the latest version of the Playful Sparkle - SEO Pack extension from the link provided on this page.
3. Upload the Extension Files:
    1. Log in to your OpenCart admin panel.
    2. Navigate to Extensions > Installer.
    3. Click the "Upload" button and upload the file ps_seo_pack.ocmod.zip.
4. Install the Extension:
    1. After uploading, click the green "Install" button.
    2. Go to Extensions and select Modules from the dropdown menu.
    3. Locate the "Playful Sparkle - SEO Pack" extension in the list.
    4. Click the green "Install" button to activate it.
5. Configure the Extension:
    1. Stay on the Extensions page with Modules selected.
    2. Click the "Edit" button next to the installed Playful Sparkle - SEO Pack extension.
    3. Configure the extension as desired, enabling or disabling Structured Markup Data, Dublin Core, Open Graph, and Twitter.
    4. Save your configuration.

Troubleshooting

- Structured Data not displaying: Ensure Structured Markup Data is enabled in the settings and that your product data is correctly filled out.
- Metadata tags not appearing: Verify that the corresponding Dublin Core, Open Graph, or Twitter options are enabled in the settings.
